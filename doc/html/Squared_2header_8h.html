Перечисления

◆ MARGIN_TYPE

enum MARGIN_TYPE

enum для типа погрешности.

Данный тип помогает различать типы погрешностей. Вычисляются погрешности дискриминанта и вещественной части корней уравнения.

◆ STATUS

enum STATUS

enum для типа комплексного числа.

Данный тип содержит в себе информацию о том есть ли у комплексного числа вещественная или мнимая части.

◆ TEST_RESULTS

enum TEST_RESULTS

enum для результата юнит-тестирования.

Данный тип информирует о результатах юнит-тестирования, а также сообщает тип ошибки в тесте.
